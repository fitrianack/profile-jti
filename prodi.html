    PROGRAM STUDI D3 TEKNIK INFORMATIKA

    VISI

    "Menjadi program studi yang unggul di bidang Teknologi informasi dan menghasilkan tenaga profesional yang bermoral, mandiri, serta mampu bersaing di dunia industri."

    MISI

  • Menyelenggarakan program pendidikan di bidang Teknologi informasi berbasis industri.
  • Menjalin kerjasama dengan pihak stakeholder serta melaksanakan pengabdian masyarakat secara simultan.
  • Melakukan penelitian dan pengembangan keilmuan di bidang Teknologi informasi dari permasalahan yang terjadi di dunia industri.

    • PROFIL LULUSAN

  • Pemrogram aplikasi (Programmer Application)
  • Merancang aplikasi dan basis data (Designer Application and database)
  • Admin Jaringan (Networking Administrator)
  • Pendukung kebutuhan IT perusahaan/industri (IT Support)
  • Intrapreneur / Technopreuner

    PROGRAM STUDI D4 REKAYASA PERANGKAT LUNAK

    VISI

    "Menjadi program studi yang unggul di bidang Software Engineering tingkat nasional dan berdaya saing global."

    MISI

  • Meningkatkan mutu, akses, dan relevansi pendidikan Politeknik di bidang Software Engineering untuk menghasilkan lulusan sesuai kebutuhan pekerjaan
  • Melakukan penelitian terapan dan pengabdian masyarakat di bidang Software Engineering untuk mengatasi persoalaan industri dan masyarakat

    • PROFIL LULUSAN

  • System Analyst : mampu menganalisis dan menspesifikasikan kebutuhan perangkat lunak berskala besar dengan dokumentasi yang mengacu standar IEEE Std 829-1998 atau standar lain.
  • Software Developer : mampu mengembangkan perangkat lunak dengan menerapkan metodologi pengembangan perangkat lunak yang sesuai dengan perkembangan terkini agar menghasilkan perangkat lunak yang sesuai dengan kebutuhan pengguna.
  • Software Tester: mampu melakukan pengujian perangkat lunak menggunakan metode black box dan atau white box sesuai kebutuhan.
  • Software Maintenance: mampu melakukan perawatan perangkat lunak dari tingkat sederhana sampai dengan kompleks.
  • Technopreneur : Mempu menginternalisasi semangat kemandirian kejuangan dan kewirausahaan, serta mampu menunjukkan sikap bertanggung jawab, jujur, dan disiplin atas pekerjaan dibidang keahliannya secara mandiri.
